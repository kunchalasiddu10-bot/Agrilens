About AgriLens

AgriLens is an AI-powered Crop Disease Detection system designed for the Project Expo at Geetanjali College, Nellore. Built by the CSE Department at PBR VITS Kavali, this project aims to bridge the gap between advanced technology and agriculture.

Key Features

  • Detects over 100+ crop diseases
  • Modern AI simulation engine
  • Provides scientifically accurate treatments
  • Built with Python, Flask, and Premium UI
